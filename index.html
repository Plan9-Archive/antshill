Advanced Namespace Tools for Plan 9

9front-ANTS live/install CDs available

Free public 9p service grid online

ANTS is a collection of modifications and additional software which adds new namespace manipulation capabilities to Plan 9. It is free software based on 9front and uses the same licensing, MIT for original code, LPL for modifications of Bell Labs source.

Public 9p service grid

  • Services: chat, wiki, ramfs, rootfs, plumber, registries.
  • Usable from any 9p-capable client. ANTS contains grid tools but ANTS is not required.
